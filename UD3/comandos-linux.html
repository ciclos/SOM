Comandos Linux

A continuación se listan los comandos mas comunes cuando se administra Linux desde la línea de comandos.

ls

Lista el contenido del directorio donde se invoca

cd [nombreDirectorio]

Entra dentro del directorio llamado "nombreDirectorio". Si queremos volver al directorio que contiene al directorio donde estamos ( el directorio padre) escribimos: cd..

cp [origen] [destino]

Copia un archivo desde la ruta de "origen" en la ruta "destino"

rm

Borra un archivo. Para borrar un directorio usaremos: rm -R

mv

Mueve un archivo de una ubicación a otra. También nos puede servir para renombrar un archivo o directorio.

clear

Limpia la pantalla.

cat

Muestra el contenido de un archivo.

tree

Muestra el contenido del directorio en forma de arbol.

grep

El comando grep busca dentro de un archivo las líneas que contengan el patrón que se le indica por parámetros.

cut

El comando cut permite, como su nombre indica, cortar de una cadena de texto, las partes que nos interesen. Tiene varias opciones pero las mas útiles son -c y -f. La primera para cortar una serie de caracteres específicos y la segunda para cortar campos. En esta segunda opción debemos indicar cual es delimitador, es decir, que indica que termina un campo y empieza el siguiente, suele ser una coma o un espacio.

Supongamos que tenemos un archivo llamado nombres.txt, con el siguiente contenido:

Francisco Garcia Perez
Fernando Camuñas Leandrez
Federico Albacete Galan

Si invocamos el siguente comado:
cut -f 2 -d " " nombres.txt
obtemos la siguiente salida:

Garcia
Camuñas
Albacete

Si invocamos el siguente comado:
cut -c 2,3,4 nombres.txt
obtemos la siguiente salida:

ran
ern
ede

tr

Este comando permite cambiar unos caracteres por otros. Aunque normalmente una de sus funciones mas usuales en la administración de sistemas es comprimir los espacios, es decir, quitar los espacios entre palabras para dejar solo uno.

Por ejemplo, el comando :
echo "Hola como estas?" | tr -s ' '
daría como resultado:
Hola como estas?

Esto es muy útil cuando se usan otras herramientas, como cut, donde se debe indicar un delimitador, pues no es lo mismo un espacio que varios. Por ejemplo,
echo "Hola como estas?" | cut -f 2 -d " "
no nos daría el resultado esperado, a no ser que hagamos:
echo "Hola como estas?" | tr -s ' ' |cut -f 2 -d " "

tail

Este comando presenta por defecto las ultimas 10 lineas de un archivo. Suele ser muy útil para consultar archivos de configuración o de logs del sistema.

Por ejemplo, supongamos que acabamos de añadir un usuario al sistema y queremos comprobar si ha salido bien, podemos usar el comando de la siguiente forma: tail /etc/passwd. O podemos consultar las ultimas entradas del log del sistema: tail /var/log/syslog.
