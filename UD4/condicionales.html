Condicionales en Bash

Condicionales

Un condicional es una estructura de control que permite realizar una acción solo si se cumplen ciertas condiciones.
En Bash los condicionales se escriben de la siguiente forma: if [[ condicon ]]
then
... acciones
fi
Las acciones pueden ser cualquier codigo valido en bash y la condicion se puede forma con los operandos que aparecen en los siguientes apartados.

Operadores de comparación

Para comparar numeros enteros tenemos:

  • $a -eq $b Cierto si el contenido de a es igual al de b.
  • $a -ne $b Cierto si el contenido de a no es igual al de b.
  • $a -gt $b Cierto si el contenido de a es mayor al de b.
  • $a -lt $b Cierto si el contenido de a es menor al de b.
  • $a -ge $b Cierto si el contenido de a es mayor o igual al de b.
  • $a -le $b Cierto si el contenido de a es menor o igual al de b.

Para comparar cadenas de caracteres tendriamos:

  • $a == $b. Será cierto si las cadenas son iguales.
  • $a != $b. Será cierto si las cadenas son distintas.

Podemos también crear condiciones sobre las propiedades de los ficheros:

  • -e f1 Si existe el fichero f1
  • -r f1 Si f1 tiene permiso de lectura
  • -w f1 Si f1 tiene permiso de escritura
  • -x f1 Si tiene permiso de ejecución
  • f1 -nt f2 Si f1 es mas nuevo que f2
  • f1 -ot f2 Si f1 es mas viejo que f2
  • ! Negación. Invierte el sentido del test

Condicional compuesto

Un condicional compuesto realiza unas acciones si la condicion es cierta y otras si la condicion es falsa. En Bash se escribe de la siguiente forma: if [[ condicon ]]
then
... acciones
else
... otras acciones fi
Podemos ir mas allá anidando condiciones dentro de otras, como las siguientes: if [[ $a > $b ]]
then
     echo "mayor"
elif [[ $b > $c ]]
then
     echo "lo que sea"
else
     echo "nada de lo anterior"
fi
