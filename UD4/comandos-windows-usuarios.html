Comandos Windows para gestión de usuarios

net user

Este comando permite gestionar los usuarios desde la línea de comandos.
Si escribimos el comando, sin mas, nos muestra los usuarios del sistema.
En cambio, para añadir un usuario nuevo escribiriamos: net user nombre_usuario password /ADD Para eliminar un usuario el comando es muy similar, como se muestra a continuación: net user nombre_usuario password /DELETE

net localgroup

Permite crear grupos y añadir usuarios al mismo. Para añadir un grupo hariamos lo siguiente: net localgroup nombre_grupo /add Seguidamente para añadir usuarios al grupo: net localgroup nombre_grupo nombre_usuario /add Para saber los usuarios incluidos en grupo: net localgroup nombre_grupo Para borrar el grupo: net localgroup nombre_grupo /delete Para sacar un usuario de un grupo: net localgroup nombre_grupo nombre_usuario /delete
