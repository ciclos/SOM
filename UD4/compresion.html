Compresion

Comprimir un archivo o un directorio consiste en crear un nuevo archivo que contiene toda la información del archivo o directorio, pero en un formato que permite almacenar esos datos usando menos bits. Esto es posible porque los formatos internos que se usan los ficheros para almacenar la información creada por el usuario contienen datos que pueden ser redundantes, es decir, repetidos. Por ejemplo, simplificando mucho, imaginemos una imagen compuesta un paisaje donde el cielo es azul por completo, en un solo tono de azul. Ese cielo está compuesto por muchos "puntos" azules, todos iguales, imaginemos que un millón, por lo que en la imagen se almacena un millón de veces el punto azul. La compresión en este caso consistiría en crear un nuevo archivo donde se indique que debe repetir un millón de veces el punto azul en vez guardar un millón de puntos azules.

Dependiendo del algoritmo de compresión que se utilice podemos tener mejores resultados de compresión, dependiendo también de que tipo de archivo sea el original. Los algoritmos de compresión mas comunes son: zip, rar, tar y gz.

Compresión en linux

Todos los linux traen una herramienta de compresión en linea de comandos llamada tar. Como todos los comandos tiene multitud de opciones, pero basicamente necesitaremos los siguientes.
Para poder empaquetar ficheros, utilizamos el siguiente comando: tar -cvf archivo.tar /dir/a/comprimir/

  • -c : indica a tar que cree un archivo.
  • -v : indica a tar que muestre lo que va empaquetando.
  • -f : indica a tar que el siguiente argumento es el nombre del fichero.tar.
Para poder desempaquetar los ficheros .tar, utilizamos el siguiente comando: tar -xvf archivo.tar
  • -x : indica a tar que descomprima el fichero.tar.
  • -v : indica a tar que muestre lo que va desempaquetando.
  • -f : indica a tar que el siguiente argumento es el nombre del fichero a desempaquetar.
Podemos usar la opción -C para extraer los ficheros a un directorio especifico: tar -xvf archivo.tar -C /directorio/donde/extraer Si se quiere ver el contenido de un fichero .tar, se utiliza el siguiente comando: tar -tf archivo.tar
  • -t : Lista el contenido del fichero .tar
  • -f : indica a tar que el siguiente argumento es el nombre del fichero a ver.
